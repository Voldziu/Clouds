O aplikacji Maklowicz

Aplikacja Maklowicz PWA została stworzona jako projekt laboratoryjny z przedmiotu "Programowanie w chmurze", inspirowana niesamowitymi przygodami kulinarnymi Roberta Makłowicza.

Czym jest PWA?

Progressive Web App (PWA) to aplikacja internetowa, która łączy w sobie zalety tradycyjnych stron internetowych i aplikacji natywnych. PWA działa w przeglądarce, ale oferuje funkcje podobne do aplikacji natywnych, takie jak:

  • Działanie offline
  • Szybkie ładowanie
  • Możliwość instalacji na urządzeniu
  • Powiadomienia push
  • Dostęp do interfejsów urządzenia

Technologie wykorzystane w projekcie

Aplikacja została zbudowana przy użyciu następujących technologii:

  • HTML5
  • CSS3 (z animacjami)
  • JavaScript (VanillaJS)
  • Service Workers
  • Web App Manifest
  • Push API
